Javascript Basics

Variables

Variables in JavaScript are written in the following form.

var [name] = [value]

"name" can be anything you want to name it.

"value" can be something like a number, a function, a string/text, or true/false.

Variables are used to store values that can be used for later.

Functions

Functions are written like this.

function [name]() {
[logic]
}

"name" can be anything you want to name it.

"logic" is where the logic of your function goes.

Functions are used for making a part of your website do something, particularly buttons.

Applying to HTML

Getting our JavaScript to our HTML requires some HTML code.

<script src="[file location]"></script>

"file location" is the location of your JavaScript file. If it is in the same folder as your HTML file then just write the name of your file. If it is in another folder then add "../[folder name]/[file name]".

We add this line inbetween our <head> tags to get our JavaScript file to be read.

<button onclick="[function name]()"> [button name] </button>

"function name" is the name of the funciton you want to use in your JavaScript file.

"button name" is the text that will be shown inside your button.

This is the code we would need for our button that runs our JavaScript code.
