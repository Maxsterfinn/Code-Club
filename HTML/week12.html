Javascript Basics

DOM

DOM stands for "Document Object Model" and it is a structure that allows us to access HTML and XML documents and treat them like objects. This means we can change HTML and XML files through DOM and it includes adding tags, changing the style(CSS), and more things.

document.getElementsByTagName("[tag name]");

This is an example of DOM being used in Javascript. In this case, it is being used to get a tag. We can store this as a variable, then we can do other things with this tag like change the CSS or add elements/tags after it.

document.getElementById("[tag id]");
document.getElementsByClassName("[tag class]");
document.createElement("[tag name]");

These are some basic ways to interact with DOM by getting a certain tag. What do we do when we have this tag?

var [name] = document.getElementById("[tag id]");

Well first let's set it to a variable. Now we can call the variable and try to change it.

[name].[attribute] = "[attribute value]";
[name].setAttribute = ("[attribute]","[attribute value]");
[name].style.[attribute] = "[attribute value]";
[name].innerHTML = "[text]";
[name].textContent = "[text]";

These lines are examples of how we can change the CSS and the content inside tags with JavaScript.

[name].parentNode.insertBefore([tag name], [name]);

In order to insert a tag before another one, you would use the line above. "name" is the tag that you want to insert before and "tag name" is the tag you want to insert. If you want to insert something after then in the second "name" add a ".nextSibling".

[name].parentNode.insertBefore([tag name], [name].nextSibling);
